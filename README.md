## Beat

## Overview

Beat is a user-friendly application designed to provide a simple yet modern countdown timer. It allows users to set countdown timers and save them locally.

## Installation

```bash
git clone https://github.com/hoangvu12/beat
cd beat
pnpm i
pnpm dev
```
